# Workout-Updater
## Purpose
Updates Notion table to progressively add weight and reps to specific exercises.
- Pulls in workouts from the past week
- Adds reps if status checkbox is ticked and current reps is under 12
- Adds 5 lbs and resets reps to 8 if status checkox is ticket and current reps are at 12
